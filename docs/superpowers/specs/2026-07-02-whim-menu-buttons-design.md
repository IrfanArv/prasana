# Whim Menu Buttons Design

Replace the existing “Our Menu” button in the Whim section with three buttons:
“Breakfast Menu”, “Lunch & Dinner Menu”, and “Beverage Menu”. Keep the existing
“Reserve Your Table” button.

Each menu button reuses the website’s existing `btn-book-header` styling and
opens its corresponding local PDF in a new tab with `rel="noopener"`. The PDFs
live in `public/img/dining/`. No navigation submenu or new styling is needed.
